    Explain block scope. Is "let" block-scoped?

    Block-scoped means that the let exists in the function, but has no effect to anything else in the program. Let is block-scoped because it can’t be accessed outside of the function.

    What is a spread operator? Why is it useful?

    Allow you to work with arrays and lists of numbers. It spreads out the array into a list of numbers that can be used by a Math.max for example.

    Code Example

    This is useful because it allows you to flexibly transform arrays without having to write any complicated loops.

                
                const numbers = [1, 10, 99, -5];

                console.log(Math.max(...numbers)); //returns 99

    How does the compiler determine if the triple dots is a spread operator or a rest operator?

    It depends on where you use the triple dots. If it is used to combine variables in the function then it is the rest operator. If it is used when you call the function then it will be used as the spread operator.
